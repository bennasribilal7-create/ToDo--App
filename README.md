# ToDo--App
Eine einfache terminalbasierte ToDo-App in Python. Das Programm ermöglicht das Hinzufügen, Anzeigen und Löschen von Aufgaben. Alle Einträge werden mit Zeitstempel versehen und dauerhaft in einer JSON-Datei gespeichert. Einsteigerprojekt zum Arbeiten mit Dateien, JSON und grundlegender OOP in Python.
