# ToDo--App
ToDo App (Python)

Eine einfache, terminalbasierte ToDo-App in Python. Sie ermöglicht das Erstellen, Anzeigen und Abhaken von Aufgaben. Alle Einträge werden in einer JSON-Datei gespeichert, sodass sie nach dem Beenden des Programms erhalten bleiben.
Funktionen

Aufgaben hinzufügen
Neue ToDos werden mit einem Zeitstempel versehen und gespeichert.

Aufgaben anzeigen
Alle gespeicherten Aufgaben werden nummeriert ausgegeben.

Aufgaben abhaken
Durch Eingabe der entsprechenden Nummer kann eine Aufgabe gelöscht werden.

Persistente Speicherung
Die Daten werden in tasks.json abgelegt und beim Start automatisch geladen.

Technologien

Python 3

JSON zur Datenspeicherung

datetime für Zeitstempel
